# Issue and Exception Log

AI Web Design System v0.1 (Pilot). Running log of issues and exceptions during the pilot. Log anything worth learning from, good or bad. This feeds the pilot review (11).

## Type Legend

Use one of these fixed types in the Type column.

| Type | Meaning |
|------|---------|
| AI output rejected | AI produced something not fit to use |
| Human rewrite | A person rewrote AI output substantially |
| Workflow bypassed | A defined step was skipped |
| Design-system gap | Missing token, component, or rule |
| Breakdance manual correction | Manual fix needed in Breakdance |
| AI lacked context | AI failed for want of information in the context pack |
| Big time saving | AI saved notable time, worth recording |
| Governance delay | Approval or gate slowed the work |
